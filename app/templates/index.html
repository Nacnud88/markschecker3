How to Use - High Volume Edition

Getting Started:
  1. Log in to your Voila.ca account
  2. Select a pickup time in the region you are searching for
  3. Open Developer Tools (press F12), go to the Application tab
  4. In the left sidebar, expand Cookies and click on voila.ca
  5. Look for "global_sid" cookie and copy its value
  6. Paste the session ID in the input field
  7. Enter your search terms (article codes or general product names)
  8. Click "Search Products"
High Volume Processing

This enhanced version can handle up to 40,000 articles efficiently:

  • Chunk Processing: Articles are processed in batches of 500 for optimal performance
  • Database Storage: Results are temporarily stored in SQLite for efficient handling
  • Progress Tracking: Real-time progress indicators show chunk and overall progress
  • Memory Optimization: Efficient memory usage prevents system overload
  • Auto Cleanup: Data is automatically cleaned up when you leave the page
Article Search

Use when you have specific product codes. The tool automatically detects formats like:

  • Comma-separated: 123456EA, 789012EA
  • One code per line
  • Continuous list: 123456EA789012EA

Performance: Can handle tens of thousands of article codes in a single search.

Generic Search

Use when searching for general terms like "Water" or "Coffee".

Choose how many results you want per term (up to 50).

Note: Generic searches with many terms may generate very large result sets.

Processing Time: Large searches (1000+ articles) may take 5-15 minutes depending on size and server load.
Important: Your session ID is never stored or shared, but using this tool will count as activity on your Voila account.
